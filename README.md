# Medicine-delivery-app
# Medicine Delivery Web Application

A full-stack medicine delivery web application that allows users to order medicines from nearby pharmacies and receive doorstep delivery quickly and securely.

---

## Features

- User authentication and login system
- Browse and order medicines online
- Pharmacy and medicine management
- Responsive user interface
- Secure backend API integration
- Order management functionality
- Planned real-time order tracking

---

## Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL
